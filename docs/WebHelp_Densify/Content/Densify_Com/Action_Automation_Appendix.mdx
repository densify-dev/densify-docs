Prior to installing or upgrading the PowerCLI version for your Densify Connector, please check the following compatibility list. Running incompatible versions may result in Action Automation failing to execute. Also see [vCenter and PowerCLI Version Compatibility](./Action_Automation_Appendix_B) for a matrix of which versions are supported with each version of vCenter.

In general you want the latest version of the Densify Connector that matches your requirements.
