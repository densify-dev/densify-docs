1. Select the system and then click the Reference tab > Analyses button.
2. The list of analyses that have been run fo the selected system are displayed.
3. Click the type of analysis you want to review.
4. Click on the map icon in the resulting report.
5. In the Overall Compatibility Map, click Analysis tab > Display option to show only the scores used to determine why the recommendation was made.
6. Double-click the source-target intersection to see Overall Compatibility Report or Overall Compatibility Report (Cloud).

The Overall Compatibility Report provides a detailed summary on the intersection of a source and target system. The report provides a ruleset summary on the system configuration compatibility and a workloads summary on the workload stacks.

1. Select the Overall Compatibility Map.
2. Double-click at an intersection of a source and a target. The Overall Compatibility report will be generated and displayed.
