## Prerequisites for the Analysis Workbench

Before you proceed, it is important that the following prerequisites are in place:

* Data has been collected and has been loaded to the Densify database.

* Data collection and analysis packages have been loaded using.

* Users have been assigned the appropriate user role so that they have the required permissions to perform their tasks.

Contact [Support@Densify.com](mailto:Support@Densify.com) if you are new to Densify.
