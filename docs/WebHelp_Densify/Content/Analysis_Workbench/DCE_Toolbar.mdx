### DCE Toolbar

This table identifies the various Data Center Explorer toolbar options.

You can also access features and functionality by right-clicking on a folder in the tree viewer. This right-click functionality, allows you customize your tree view using view and filter options and then, view details or create analyses based on the subset of systems.
