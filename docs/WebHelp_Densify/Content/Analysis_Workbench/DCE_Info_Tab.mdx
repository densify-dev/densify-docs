### DCE Info Tab

This tab provides general, technical, business, and workload information for either a selected folder or a single system. The folder can be at the platform, domain, datacenter, cluster or host level.

<Note>
Some features are only available for single systems and are not displayed in the toolbar when a folder is selected; while other features are for a folder and not available for a single selected system.
</Note>
