title: "Shortcut Keys"
---

You can use the shortcut or function keys to quickly access some of the Control Console functionality.

Table: Function Keys

<table>
<thead>
<tr>
<td>
<p>Key</p>
</td>
<td>
<p>Function</p>
</td>
<td>
<p>Description</p>
</td>
</tr>
</thead>
<tr>
<td>
<p><span >F2</span></p>
</td>
<td>
<p>Toggle <span class="mc-variable Document.DropZone variable">drop zone</span> On/Off</p>
</td>
<td>
<p data-mc-conditions="CiRBA.SaaSOnly">Opens/closes the <span class="mc-variable Document.DropZone variable">drop zone</span>. From the <span class="mc-variable Document.DropZone variable">drop zone</span> you can tag systems, create action plans, show system details, grant exemptions, create outbound bookings and create transform scenarios. </p>
</td>
</tr>
<tr>
<td>
<p><span >F4</span></p>
</td>
<td>
<p>Toggle Reveal On/Off</p>
</td>
<td>
<p>Opens/closes the <span >Reveal</span> dialog box.</p>
</td>
</tr>
<tr>
<td>
<p><span >F7</span></p>
</td>
<td>
<p>Switch to Previous Timeframe on the Timeline</p>
</td>
<td>
<p>Moves the slider to the previous timeframe date along the timeline from the currently selected timeframe date. </p>
<p class="CiRBATableNote" data-mc-autonum="<b>Note: </b>"><span class="autonumber"><span><b>Note: </b></span></span>F7 also invokes IE caret browsing functionality, which can be disabled the first time the pop-up is displayed.</p>
</td>
</tr>
<tr>
<td>
<p><span >F8</span></p>
</td>
<td>
<p>Switch to Next Date on the Timeline</p>
</td>
<td>
<p>Switches the timeframe date to the next date from the currently selected timeframe date.</p>
</td>
</tr>
<tr>
<td>
<p><span >F9</span></p>
</td>
<td>
<p>Filter Selected System</p>
</td>
<td>
<p>Filters the Spectrum based on the one selected infrastructure group, host or VM. </p>
</td>
</tr>
<tr>
<td>
<p><span >F10</span></p>
</td>
<td>
<p>Reset Filter</p>
</td>
<td>
<p>Turns off the currently applied filter. </p>
</td>
</tr>
</table>


Table: Ctrl Key Shortcut Combinations

<table>
<thead>
<tr>
<td>
<p>Key</p>
</td>
<td>
<p>Function</p>
</td>
<td>
<p>Description</p>
</td>
</tr>
</thead>
<tr>
<td>
<p><span >Ctrl+F10</span></p>
</td>
<td>
<p>Reset Reveal and Filter</p>
</td>
<td>
<p>Turns off the currently applied filter and any <span >Reveal</span> options.  </p>
</td>
</tr>
<tr>
<td>
<p><span >Ctrl+<br/>Shift+<br/>F10</span></p>
</td>
<td>
<p>Reset <span class="mc-variable Document.Control_Console variable">Control Console</span></p>
</td>
<td>
<p>Performs a full <span class="mc-variable Document.Control_Console variable">Control Console</span> reset. This results in the following:</p>
<ul>
<li><span >Filter</span> and <span >Reveal</span> will be reset.</li>
<li><span >Timeline</span> will be set to Today and <span >Timeline</span> pane will be collapsed.</li>
<li><span >Display with Recommendations Applied</span> will be set to No.</li>
<li><span >Spectrum Detail</span> pane will be collapsed.</li>
<li><span >Inbound/Outbound</span> panes will be collapsed.</li>
<li><span >Environment Navigator</span> will be expanded.</li>
</ul>
</td>
</tr>
<tr>
<td>
<p><span >Ctrl+1</span></p>
</td>
<td>
<p>Select all hosts</p>
</td>
<td>
<p>Selects all host systems in the displayed Spectrum. </p>
<p>If you have selected an infrastructure group, then pressing <span >Ctrl+1</span> selects all hosts in that infrastructure group. Pressing <span >Ctrl+1</span> again, selects all hosts in the environment. </p>
</td>
</tr>
<tr>
<td>
<p><span >Ctrl+2</span></p>
</td>
<td>
<p>Select all VMs</p>
</td>
<td>
<p>Selects all VM systems in the displayed Spectrum. </p>
<p>If you have selected an infrastructure group, then pressing <span >Ctrl+2</span> selects all VMs in that infrastructure group. Pressing <span >Ctrl+2</span> again, selects all VMs in the environment. </p>
</td>
</tr>
<tr>
<td>
<p><span >Ctrl+A</span></p>
</td>
<td>
<p>Select all systems in the Spectrum</p>
</td>
<td>
<p>Selects all hosts and VMs in the displayed Spectrum. </p>
<p>If you have selected an infrastructure group, then pressing <span >CTRL+A</span> selects all hosts and VMs in that infrastructure group.</p>
<p>In a list view, pressing <span >CTRL+A</span> selects all items in the displayed list.</p>
</td>
</tr>
<tr>
<td>
<p><span >Ctrl+></span></p>
<p>
<span >
`Ctrl+<`
</span></p>
</td>
<td>
<p>Increase/Decrease size of select dots in the Spectrum</p>
</td>
<td>
<p><span >CTRL+></span> increases the size of selected dots in the Spectrum. This size specification is used when dots are selected for view or when dragging to the drop zone.</p>
<p><span >`CTRL+<`</span> reduces the size to the default setting.</p>
</td>
</tr>
</table>
