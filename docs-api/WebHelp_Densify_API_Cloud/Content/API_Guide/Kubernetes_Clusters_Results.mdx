---
title: "Kubernetes Clusters Results"
---

This resource is used to return container recommendations at the cluster level. This endpoint shows all analyzed containers in the specified cluster.

## Resources

* `/kubernetes/clusters/<clusterName>/containers`
* `/analysis/containers/kubernetes/clusters/<clusterName>/containers?details=true`
