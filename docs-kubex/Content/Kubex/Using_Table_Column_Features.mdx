Each tabular reports provides a number of powerful features that enable you to review details on one or more systems to investigate issues highlighted by the Kubex analysis or to better understand how Kubex's recommendations can improve the health of your environment and save you money.

## Creating Filters

You can create filters in many ways. Creating filters using the options in each column is the easiest way to quickly see the results. See [Using Filters](/docs-kubex/Content/Kubex/Using_Filters) for details.

You can also create filters using the features in the tree viewer. See [Creating Custom Filters](/docs-kubex/Content/Kubex/Creating_Custom_Filters).

## Adding or Removing Columns

Expand the flyout menu on the right side of the page to see all of the available columns:


<ol start="1">
  <li>All of the columns available for the selected table are displayed in this menu.</li>
  <li>Select or de-select columns to obtain the desired data set.</li>
</ol>


The table updates as you select or de-select columns in the flyout menu.

## Creating Other Views

You can configure the table to display only the columns of interest. 


<ol>
  <li>Access the column selection dialog box, from each column's context menu or from the icon on the right side of the page.</li>
  <li>Check the columns you want to see in your tabular view. Deselect the columns you do not want to see.</li>
  <li>You can search for a column. As you type the search criteria the list updates to show only the matching items.</li>
  <li>As you select columns, the table updates to show the selected columns.</li>
  <li>You can export your customized view.</li>
  <li>The selected table view settings are re-applied when you navigate away from this page.</li>
</ol>

## Exporting Data

You can export tabular reports from this tab to Microsoft® Excel.


<ol>
  <li>Select the table view and adjust views and filters to obtain the desired set of data.</li>
  <li>Click the export icon, located beside the Table View drop down menu.</li>
</ol>


Figure: Export Button
![](/images/docs-kubex/Content/Kubex/03000017.png)
When the content of the AI Analysis Details is exported, entity names that are provided as a [hyperlink](/docs-kubex/Content/Kubex/Visualizing_Your_Data#deep-links) are included in the exported XLS file. When you open the exported .XLS file, the URL content initially appears as plain text, and when you click on the cell it becomes a hyperlink.

## Moving Columns

Once you have all of the required columns in your table, you can move the columns around to again obtain the desired view.
Click on the header of the column and drag it left or right to the desired location.

## Context Menu

Right-click on any cell in the table to open a context menu with the following options.


<InlineImageWithText width="750" src="/images/docs-kubex/Content/Kubex/03000216_284x245.png">


<ul>
  <li>Copy--Copy the value.</li>
  <li>Copy with Headers--Copy the value, including the column header.</li>
  <li>Copy with Group Headers--Copy the value, including the an indication that the selected column has been used to group data.</li>
  <li>Chart Range--These options are not applicable as only a single item can be selected.</li>
  <li>Export--Export the selected content to CSV or Excel format. If you are exporting a value with a hyperlink, the hyperlinked URL is not exported when using .CSV format</li>
  <li>Cut/Paste--These options are not applicable.</li>
</ul>
