The cards on this tab provide a visual overview of a single node group. When navigating the tree viewer, you will see this tab when a single node group is selected. This page is different from the [Node Overview](/docs-kubex/Content/Kubex/Node-Overview-Tab) page that is displayed when a single node is selected.

You will see the [Node Group Analysis Tab](/docs-kubex/Content/Kubex/Node_Group_Analysis_Table) tab when more than one mode group is selected.


<Frame caption="Figure: Locating the Node Group Overview">
![](/images/docs-kubex/Content/Kubex/03000143_661x199.png)
</Frame>



## Analysis Summary

Click on any of the listed settings to see a brief description. Complete descriptions of each setting are provided in the [Node Group Analysis Tab](/docs-kubex/Content/Kubex/Node_Group_Analysis_Table) tab.
Identifying details and the recommendation are displayed on this card.

<div style={{ width: "100%", display:"flex", flexDirection:"row", alignItems:"center", gap:"20px"}}>
  <Frame caption="Figure: Analysis Summary Details">
    ![](/images/docs-kubex/Content/Kubex/03000144_545x195.png)
  </Frame>
  <ul style={{ width: "50%"}}>
    <li>
      <Tooltip tip="Indicates the reason that more containers cannot be added to this node group.">
      Primary Constraints
      </Tooltip>
    </li>
    <li>
      <Tooltip tip="This is the estimated wasted spend per month. If the value is negative your infrastructure may be under-provisioned.">
      Estimated Monthly Waste
      </Tooltip>
    </li>
    <li>
      <Tooltip tip="The current cost of running this node. The calculation is based on whether the primary node type has been defined.">
      Current Monthly Cost
      </Tooltip>
    </li>
    <li>
      <Tooltip tip="The cost of running this node on the optimal instance type. The calculation is based on whether the primary node type has been defined.">
      Fully Optimized Cost
      </Tooltip>
    </li>
    <li>
      <Tooltip tip='One of the following: “Compute Optimized", “General Purpose”, or “Memory Optimized”. If the node is not one of these, then this value cannot be determined and is indicated as a dash (-).'>
      Node Family
      </Tooltip>
    </li>
    <li>
      <Tooltip tip="Use this value to determine if your nodes are utilized as efficiently as possible. Both CPU and memory utilization are used to determine this ratio.">
      Fully Optimized Node Family
      </Tooltip>
    </li>
  </ul>
</div>

    Node Balance Ratio

## Utilization Charts

The last section provides utilization charts that show hourly min/max and sustained activity for the selected node group. You can also expand any chart to the modal view and select other views. See [Analysis Details Table](/docs-kubex/Content/Kubex/AI_Analysis_Details_Table) for details on using these charts to review workload data.
