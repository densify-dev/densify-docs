## Initiating Container Data Collection



<ContentWrapper src="/images/docs-kubex\Content\Kubex/03000115.png">
The Densify data forwarder is a container that collects data from Kubernetes using the Prometheus server and then forwards that data to Kubex.<br/>
<br/>
You can quickly deploy the data forwarder and all of the required prerequisite software using a Helm chart. See <a href="https://github.com/densify-dev/helm-charts/tree/master/charts/kubex-automation-stack">Kubex Automation Stack Helm Chart</a>.<br/>
<br/>
Other configuration options including multi-cluster and observability platforms are also supported. See <a href="/docs-kubex/Content/Kubex/Data_Collection_Overview" xrefformat="{u}{paratext}{/u}">Data Collection for Containers</a> or <a href="https://github.com/densify-dev/container-data-collection/blob/main/requirements.md#data-source">Github</a> for other configuration options.
</ContentWrapper>
