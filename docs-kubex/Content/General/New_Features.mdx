<Update label="July 9, 2025">
## Kubex

The following changes have been made to Kubex in this release:


<ul>
  <li>
    Updated Container Summary Tab--The container <a href="/docs-kubex/Content/Kubex/Summary_Tab"> Summary page </a> has been updated to better highlight both your current spend and the details of possible savings.
  </li>
  <li>Updated Utilization Chart--The node chart showing the number of pods running on a node has been updated to show the limit for the selected node.</li>
  <li>Customizable Data Aggregation--You now have the ability to edit how grouped data in a table is aggregated (i.e. sum, average, max, etc.) and save your customization in a table view. This feature is available for containers, node groups, nodes and cloud instances.</li>
</ul>



## Cloudex

The following features are new or updated in this version:


<ul>
  <li>
    Cloud Connections--In this release you can now can create and edit the <a href="/docs-kubex/Content/Cloudex/Cloud_Connections"> connections </a> to AWS and Azure. You can also view data collection status for each of the connections. Cloudex uses these connections to collect data from each account, daily.
  </li>
  <li>
    Accessing the Catalog Map--You can now access the catalog map directly from the <a href="/docs-kubex/Content/Cloudex/AI_Analysis_Details_Table-Cloud"> Analysis Details </a> tab. The catalog map with a modal view has been added to the utilization charts.
  </li>
  <li>
    Summary Tab--The Cloud <a href="/docs-kubex/Content/Cloudex/Summary_Tab-Cloud"> Summary </a> tab has been updated to improve useability.
  </li>
</ul>
 
</Update>

Refer to the [Densify Release Notes](https://www.densify.com/docs/WebHelp_Densify_Cloud/Content/Release_Notes/New_Features_Cloud) for a complete list of 3.x features.
